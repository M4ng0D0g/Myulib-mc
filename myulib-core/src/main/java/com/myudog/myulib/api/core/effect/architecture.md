## 特殊效果補充

- 反射: 在允許攻擊範圍內，反射者將依照比例將受到的傷害反彈給攻擊者。
- 尖刺: 在允許攻擊範圍內，尖刺者將依照比例將受到的傷害共同給攻擊者。

- 嘲諷: 在允許攻擊範圍內，被嘲諷者的任何攻擊傷害將被轉移到嘲諷者身上。
- 庇護: 在允許攻擊範圍內，庇護者將承受所有對被庇護者的攻擊傷害，直到庇護者死亡或效果結束。

- 無敵: 角色完全免疫所有傷害和負面效果。
- 淨化: 角色自動清除所有負面效果和傷害，持續期間不會被賦予負面效果。

- 攻擊禁止: 所有攻擊行為被禁止，包括近戰和遠程攻擊。
- 技能禁止: 所有技能使用被禁止，包括主動技能。
- 移動禁止: 所有移動行為被禁止，包括行走、奔跑、跳躍等。
- 轉向禁止: 所有轉向行為被禁止。
