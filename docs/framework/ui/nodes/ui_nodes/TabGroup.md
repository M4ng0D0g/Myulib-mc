# TabGroup

Path: `src/client/java/com/myudog/myulib/client/api/ui/node/TabGroup.java`

Description

Container representing a group of tabs. Works together with `TabButton` nodes to switch visible content.

Public API

- Constructor:
  - `TabGroup()`

Usage example

```java
val tg = TabGroup()
// add TabButton children and content panels
```
