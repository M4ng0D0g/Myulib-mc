# Timer models overview
`TimerModels` provides the nested timer enums, records, and instance class used by `TimerManager`.
## Current types
- `TimerMode`
- `TimerStatus`
- `TimerTickBasis`
- `TimerPayload`
- `RespawnTimerPayload`
- `SoundTimerPayload`
- `TimerAction`
- `TimerBinding`
- `TimerSnapshot`
- `Timer`
- `TimerInstance`
## Notes
- All timer callbacks flow through `TimerManager`.
