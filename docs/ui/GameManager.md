# Game system overview
`com.myudog.myulib.api.game` is the current public entry point for game registration and instance management.
## Core types
- `GameManager`
- `GameDefinition`
- `GameInstance`
- `GameConfig`
- `GameData`
- `GameState`
- `GameStateMachine`
## Notes
- Game objects live under `com.myudog.myulib.api.object`.
